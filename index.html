Tipos de Memorias Ram

Static RAM (SRAM)

La memoria estática de acceso aleatorio, también conocida como RAM estática o SRAM, es un tipo de RAM que almacena bits de datos de forma estática y retiene los datos siempre que se suministre energía. Más rápido y confiable que DRAM, se usa ampliamente en aplicaciones de electrónica, microprocesadores y computación en general.

Dynamic RAM (DRAM)

La memoria dinámica de acceso aleatorio (RAM dinámica o DRAM) es un tipo de memoria semiconductora de acceso aleatorio que almacena cada bit de datos en una celda de memoria que consta de un pequeño capacitor y un transistor, ambos típicamente basados en semiconductores de óxido de metal (MOS ) tecnología.

Synchronous Dynamic RAM (SDRAM).

La memoria de acceso aleatorio dinámica síncrona (SDRAM) es una memoria de acceso aleatorio dinámico (DRAM) con una interfaz síncrona con el bus del sistema que transporta datos entre la CPU y el concentrador del controlador de memoria. ... SDRAM espera la señal del reloj antes de responder a las entradas de control. La SDRAM precedió a la doble velocidad de datos (DDR).
